*** Trader Algorithms
    DEADLINE: <2016-09-03 sáb>
    Trend Types:
    - trending <- move neutral / bull / bear
      - HMA
      - BB channel
      - MACD
    - move strength <- play validation
      - Stochastic
      - BB above/bellow std2
    - pivot levels <- play positions lines
      - Pivot
      - Fibonacci
      - MAs
      - BB top/middle/bottom lines
    - volatility <- play type scalp / breakout
      - ATR
      - Chaikin
    - signals <- whether it shall put an order
      - HMA periods cross
      - MACD main & signal cross
      - Stochastic main & signal cross

    Algorithm Trader Structure Orchestra 01 (Simpler):
      1) onSignalTrade
      2) validates signal throught trend/strength
	 - signal vs trend/strength
      3) onVolatility set trade type
	 - onScalpTrade
	 - onBreakoutTrade
      4) using pivot levels set entry / stop / target position
	 - onScalpTradeBuy
	 - onScalpTradeSell
	 - onBreakoutTradeBuy
	 - onBreakoutTradeSell

    Algorithm Trader Structure Orchestra 02 (different signals for different volatility):
      1) onVolatility set trade type
      2) onSignalTrade <- different signals for different trade type (scalp/breakout)
	 - onSignalTradeScalp
	 - onSignalTradeBreakout
      3) validates signal throught trend/strength
	 - signal vs trend/strength
      4) using pivot levels set entry / stop / target position
	 - onScalpTradeBuy
	 - onScalpTradeSell
	 - onBreakoutTradeBuy
	 - onBreakoutTradeSell

    Algorithm Types:
    - Trend inclined (use major/current/fast trends)
    - TradingRange Helper! (signals with support&resistance levels)
    - Scalper (low volatility under fast timeframes)
    - Swinger (higher volatility & using pivot levels on slower timeframes)
    - Orchestra: macd for signals, stochastic for confirmation (overbought/oversold)
      and BB for entry/exit points.

**** Scalper algorithm

ATR (14 / 80) <- sets volatility | whether it should scalp or trend
HMA trend (20, 50 / 50, 200) <- sets trend neutral/positive/negative
BB trend (BB2 / BB3 / BB4) <- sets entry exit points
Stoch signal|main (21, 45) <- sets whether it shall pull the trigger
